# Inicializando a aplicação Meu Time

## Passo 1

Para rodar o projeto, é necessário rodar:

### `npm install`

## Passo 2

### `npm start`

## Passo 3

Abra o navegador com o endereço [http://localhost:3000](http://localhost:3000) para visualizar a aplicação rodando.
